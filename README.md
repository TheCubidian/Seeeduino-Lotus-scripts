# Seeeduino-Lotus-scripts
A small collection of arduino scripts I will be making, they will control the led, oled screen and buzzer using a combination of the button, potentiometer, and sensors (and possibly the serial port).
The exact board I am using can be found [here](https://www.seeedstudio.com/Grove-Beginner-Kit-for-Arduino-p-4549.html).
You will be able to preview real hardware functionality on the web eventually.
